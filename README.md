# 👕 Western Rise Website Clone

### Frontend Clone using HTML & CSS

---

## 🚀 Overview

This project is a **frontend clone** of the Western Rise website, built using **HTML5 and CSS3**.

The purpose of this project is to:

* Practice modern UI/UX design
* Improve frontend development skills
* Recreate a premium **e-commerce website layout**

---

## 🎯 Features

* 🎨 Clean and premium UI design
* 🖥️ Responsive layout (if implemented)
* 🧭 Navigation bar with menu links
* 🖼️ Hero/banner section
* 👕 Product listing section
* 🧾 Product cards with details
* 📄 Footer with links & info

---

## 🛠️ Tech Stack

* **HTML5** → Structure
* **CSS3** → Styling
* **Flexbox / Grid** → Layout
